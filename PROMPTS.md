# Bộ prompt NovaShop Max Premium

Bạn có thể dùng các prompt dưới đây để tiếp tục nâng cấp NovaShop thành website bán hàng cao cấp, có trải nghiệm như boutique online, tối ưu chuyển đổi và sẵn sàng triển khai production.

## 1. Prompt tổng quan dự án cao cấp

```text
Hãy đóng vai Principal Product Designer kiêm Senior Frontend Engineer. Nâng cấp website bán hàng NovaShop bằng React/Vite thành một luxury ecommerce website tiếng Việt, phong cách boutique premium, dành cho khách hàng yêu thích thời trang, công nghệ, phụ kiện, du lịch, thể thao và gia dụng cao cấp. Website cần có hero đẳng cấp, collection editorial, sản phẩm nổi bật, private sale, concierge VIP, giỏ hàng, thanh toán, đăng nhập, admin quản lý sản phẩm, responsive hoàn hảo và hiệu năng tốt. Thiết kế dùng palette navy #0d1324, gold #d8a84f, cream #fff8ea, trắng cao cấp và typography sang trọng.
```

## 2. Prompt thiết kế visual luxury

```text
Thiết kế lại UI/UX NovaShop theo chuẩn luxury ecommerce. Tạo cảm giác cao cấp bằng layout editorial, khoảng trắng lớn, ảnh hero cinematic, glassmorphism tinh tế, gold accent, shadow mềm, card sản phẩm bo góc lớn, hover motion mượt, CTA rõ ràng, sticky header, trust badges và mobile-first. Không dùng giao diện đại trà; ưu tiên cảm giác boutique, curated collection và dịch vụ cá nhân hóa.
```

## 3. Prompt nội dung thương hiệu

```text
Viết bộ nội dung brand voice cho NovaShop theo phong cách premium: headline hero, subheadline, tagline, mô tả 6 danh mục, 10 CTA, 5 đoạn mô tả dịch vụ VIP concierge, 5 câu cam kết chất lượng, 5 thông điệp đóng gói quà, 5 review khách hàng tự nhiên. Giọng văn tiếng Việt sang trọng, tin cậy, không phô trương, tập trung vào trải nghiệm mua sắm cao cấp.
```

## 4. Prompt dữ liệu sản phẩm cao cấp

```text
Tạo dữ liệu JavaScript array cho 40 sản phẩm premium. Mỗi sản phẩm có id, slug, name, category, price, oldPrice, rating, reviewCount, image, gallery, badge, description, stock, colors, sizes, materials, warranty, origin, tags, isFeatured, isLimited, giftReady. Danh mục gồm Thời trang, Công nghệ, Phụ kiện, Du lịch, Thể thao, Gia dụng. Nội dung tiếng Việt, giá VND hợp lý, ảnh dùng URL Unsplash phù hợp.
```

## 5. Prompt trang chủ max conversion

```text
Thiết kế trang chủ NovaShop tối ưu chuyển đổi: hero cao cấp, brand strip, benefits, curated luxury collections, product grid, editor's picks, flash sale/private sale countdown, VIP concierge section, reviews, FAQ, newsletter signup và footer. Mỗi section cần có CTA rõ ràng, semantic HTML, alt text, ARIA label và responsive tốt trên mobile.
```

## 6. Prompt product card cao cấp

```text
Nâng cấp card sản phẩm thành luxury product card: ảnh lớn lazy load, badge premium, quick add, wishlist, rating, review count, mô tả ngắn 2 dòng, tag Nova Select, giá cũ/giá mới, hover lift, overlay gradient nhẹ, trạng thái hết hàng và accessibility đầy đủ. Card cần đồng bộ với màu navy/gold/cream.
```

## 7. Prompt trang chi tiết sản phẩm premium

```text
Nâng cấp trang chi tiết sản phẩm NovaShop: gallery ảnh lớn, thumbnails, chọn màu/size/số lượng, thông tin chất liệu, xuất xứ, bảo hành, trạng thái tồn kho, chính sách giao nhanh, gói quà premium, sản phẩm liên quan, review khách hàng, FAQ sản phẩm và sticky add-to-cart trên mobile. Giữ trải nghiệm mượt, không phá layout hiện tại.
```

## 8. Prompt giỏ hàng và checkout cao cấp

```text
Hoàn thiện cart drawer và checkout cho NovaShop theo chuẩn premium: lưu localStorage, tăng/giảm/xóa sản phẩm, mã giảm giá, freeship threshold, gift wrapping option, delivery time slot, payment methods COD/bank/MoMo, order summary sticky, thông báo đặt hàng thành công, validation rõ ràng và UX mobile tối ưu.
```

## 9. Prompt đăng nhập và tài khoản khách hàng

```text
Thêm hệ thống account mock bằng localStorage cho NovaShop gồm đăng ký, đăng nhập, đăng xuất, hồ sơ cá nhân, địa chỉ giao hàng, lịch sử đơn hàng giả lập, wishlist và membership tier (Classic, Gold, Privé). UI phải đồng bộ phong cách luxury và có trạng thái loading/error/success.
```

## 10. Prompt admin cao cấp

```text
Nâng cấp trang admin NovaShop thành dashboard quản trị sản phẩm: thống kê doanh thu giả lập, tổng đơn, sản phẩm bán chạy, bảng sản phẩm có tìm kiếm/lọc/sort, thêm/sửa/xóa sản phẩm, preview ảnh, quản lý stock, trạng thái featured/limited/giftReady và lưu localStorage. Thiết kế dashboard rõ ràng, chuyên nghiệp, responsive.
```

## 11. Prompt SEO ecommerce

```text
Tối ưu SEO cho NovaShop React/Vite: title/meta description, Open Graph, Twitter Card, schema.org Organization, Product, BreadcrumbList, alt ảnh, cấu trúc heading, internal link, robots.txt, sitemap.xml mẫu, canonical URL và nội dung tiếng Việt hướng đến từ khóa website bán hàng cao cấp, mua sắm online, sản phẩm chính hãng.
```

## 12. Prompt hiệu năng production

```text
Audit và tối ưu hiệu năng NovaShop: React.lazy route splitting, memoization, lazy image, responsive image sizes, preconnect CDN ảnh, giảm bundle, manual chunks, CSS tối ưu, tránh layout shift, cải thiện LCP/CLS/INP, skeleton loading, accessibility audit và Lighthouse checklist mục tiêu Performance/SEO/Accessibility >= 90.
```

## 13. Prompt animation cao cấp

```text
Thêm micro-interactions cho NovaShop: hover lift card, fade/slide reveal khi scroll, CTA shimmer nhẹ, cart drawer transition, modal animation, skeleton loading và reduced-motion support. Animation phải tinh tế, không gây rối, không làm giảm hiệu năng.
```

## 14. Prompt backend production

```text
Thiết kế backend production cho NovaShop bằng Node.js/Express hoặc NestJS, PostgreSQL + Prisma, JWT auth, role admin/user, API sản phẩm/danh mục/đơn hàng/wishlist/review/coupon/upload ảnh, thanh toán MoMo/VNPAY giả lập, validation Zod, rate limit, logging, CORS, Dockerfile và migration schema.
```

## 15. Prompt deploy và vận hành

```text
Lập kế hoạch deploy NovaShop lên Netlify/Vercel và backend lên Render/Fly.io/VPS. Bao gồm biến môi trường, build command, SPA redirect, domain, HTTPS, cache headers, monitoring, analytics, backup database, checklist release, rollback plan và cách kiểm tra sau deploy.
```

## 16. Prompt A/B testing và CRO

```text
Đề xuất 10 thử nghiệm A/B cho NovaShop để tăng conversion: hero headline, CTA text, product card layout, badge trust, freeship threshold, countdown, bundle offer, checkout steps, gift wrapping upsell và exit intent offer. Mỗi thử nghiệm cần nêu giả thuyết, metric chính và cách triển khai trong React.
```
