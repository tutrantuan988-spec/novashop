# Sub-agents cho TRỌNG ĐỊNH STORE
# Inspired by everything-claude-code .agents/ directory

## E-commerce Architect
Khi cần thiết kế database, API endpoints, hoặc architecture decisions.

## Payment Integrator  
Khi làm việc với Stripe, VNPay, MoMo, webhook, refunds.

## SEO Specialist
Khi tối ưu SEO, thêm structured data, cải thiện performance.

## Firebase Admin
Khi làm việc với Firestore, Security Rules, Real-time sync.

## UI/UX Designer
Khi thiết kế giao diện, responsive, mobile optimization.

## Vietnamese Localizer
Khi xử lý tiếng Việt, i18n, currency/number formatting.
